# Anaconda Game

## Description

Become an anaconda and start eating some cows, simple and fun game created using JS.

## Usage

- Use **Enter** to start the game, and to pause it
- Use the Movement arrows to move.
- Use **R** to restart the game.

## Technologies

**HTML5, CSS3, JS**
